# Docker-LEMP

## About
Docker setup for basic LEMP stack - Linux, Nginx, MySQL, PHP.
### Versions
<ul>
  <li>Nginx - 1.20.1</li>
  <li>MySQL - 8.0.26</li>
  <li>PHP - 8.0</li>
</ul>
  
## Requirements

Install <a href="https://www.docker.com/">Docker</a>.

## Usage

Navigate to the project folder and run `docker-compose up`.
  
Visit page http://localhost:3000
